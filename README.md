# Интернет-магазин спортивных товаров (Diplom_2026)

Проект выпускной квалификационной работы: веб‑приложение интернет‑магазина спортивных товаров (инвентарь, экипировка, аксессуары).

## Технологический стек

- **Frontend**: React + TypeScript + Vite
- **Backend**: PHP (заготовка)
- **VCS**: Git

## Архитектура

Выбран подход **многоуровневой архитектуры (Layered Architecture)**:

- **Presentation layer**: `client/src` (UI, страницы, навигация)
- **Application/Business layer**: `server` (будет развиваться далее)
- **Data access layer / Database layer**: будут добавлены на следующих этапах

### Назначение макета

Макет фиксирует структуру и внешний вид ключевых экранов, а также пользовательские сценарии интернет‑магазина.

### Реализованные экраны (каркас)

- Каталог (`/`)
- Карточка товара (`/product/:productId`)
- Корзина (`/cart`)
- Личный кабинет (`/profile`)
- Админ‑панель (`/admin`)

### Пользовательские сценарии, отражённые в макете/прототипе

- Просмотр каталога и фильтрация по категории → переход в карточку товара
- Просмотр карточки товара → добавление в корзину (пока заглушка) → переход в корзину
- Просмотр корзины → переход к оформлению заказа (пока заглушка)
- Просмотр личного кабинета (профиль/история/адреса — заглушки)
- Просмотр административной панели (разделы управления — заглушки)

## Структура проекта

```
/client   — клиентская часть (React)
/server   — серверная часть (PHP, заготовка)
/public   — статические файлы (общие)
/docs     — документация
```

## Как запустить

### Клиент (React)

```bash
cd client
npm install
npm start
```

### Сервер (PHP, заглушка)

Любым способом, который поднимает `server/public` как web-root (например, встроенный сервер PHP):

```bash
cd server/public
php -S localhost:8080
```
